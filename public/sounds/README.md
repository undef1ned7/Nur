# Звук уведомлений

Поместите сюда файл `notification.mp3` — он будет проигрываться при получении
нового уведомления (см. `src/config/notificationSound.js`).

Заменить на свой звук: просто перезапишите `notification.mp3` или задайте
переменную окружения `VITE_NOTIFICATION_SOUND` с путём к своему файлу.

Если файла нет — приложение НЕ падает (звук просто не воспроизводится).
